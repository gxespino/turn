# turn - simple rotation manager for slack teams

### Setup

1. Get the code.

        % git clone git@github.com:gxespino/turn.git

2. Setup your environment.

        % cd turn

        % bundle install

3. Seed your assets and data

        % rake db:setup

3. Start the server

        % rails server
